FarmAgent is a frontend-only hackathon MVP that detects fake fertilizers using:
- AI image classification
- QR code based authenticity verification

## Features
- Fertilizer companies generate genuine QR codes
- Farmers scan QR to verify authenticity
- No backend required (static JSON verification)
- Works on web & Android (WebView)

## Tech Stack
- HTML, CSS, JavaScript
- TensorFlow.js (for AI model)
- GitHub Pages

## Future Scope
- Backend verification
- Blockchain-based QR immutability
- Government integration
